Sample flexbox example

  • flexをつけたflex-containerの子要素（flex-items）にはwidth効かない・flex container(display指定した要素)自体はblockのように盾に並ぶ
  • inline-flexをつけたflex-containerの子要素（flex-items）にはwidthきく・flex containerはinlineのように横に並ぶ
  • flex-itemsのheightはflex-containerのflex, inline-flexどちらもきく
  • flex itemsがflexになることは同じだけど、flex container がblockになるかinlineになるかが違う
