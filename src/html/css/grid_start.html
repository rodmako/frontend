Simple grid example

  • grid-gap はもうgapでいい
  • 1fr 1fr 2fr は　１行に要素が3つ入ることができて割合がflex-growのように　1 1 2の割合を持っていること
  • grid-template-columns　には　repeat(4, 1fr)　のように繰り返すこともできる
    （おな幅を持った４つの要素が１行になる）
    （1frを100frにしても有意味な動きはない）
    （frではない単位例えば、pxの場合要素がその値の分伸びて画面の外までいく）
  • grid-template-rowsを使えばなぜか1番目の行の高さしか影響がない
    grid-auto-rowsは全ての行の高さに影響がある
  • grid-auto-rows: minmax(50px, auto) - 最初50pxは保って欲しい・同時に内容が増えたら合わせて高さが伸びる（min-heightみたいな動き）
