• column-count: 2のように子要素の内容を２列で表示できる
  • column-width: 200pxのように子要素の内容を１列200pxに合わせた数の列数にすることもできる
  • column-rule: dotted or solid 6px　のように指定すれば列の間の線を引くことができる（:not(:last-child)border-left:...をしなくてもいい）
  • gap(column-gap)も使える
  • break-inside: avoid　で印刷の時要素が切れるのを防止する（もれたら次のページにいく、初期値はautoで切れる）
    page-break-insideはもうないらしい
