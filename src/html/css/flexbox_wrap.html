Sample flexbox example

  • flex containerにflex-wrapを追加したら画面に合わせてflex-itemsを改行してくれる
  • flex itemsにflex: 1; flex: 200px; などでflex items１個あたりの大きさになる（１行に収まる個数が変わる）
  • flex-direction: row, flex-wrap: wrap　は　flex-flow: row wrap;に書き換えられる
