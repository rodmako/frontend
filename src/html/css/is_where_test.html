  • :is(), :where()両方の目的は複数の要素を同時に選択するため（その中の要素を同時にスタイリングするため）
  • scssではネスト　.someClass1, .someClass2, .someClass3 { a {...} }のような書き方ができる
  • cssではそれができないため:is()と:where()が存在する
  • もし:is(), :where()がなかったら .someClass1 a, .someClass2 a, .someClass3 a { ... } のようになる。もしaのようにしエレメントセレクターではなく長い名前のクラスだとしたら...
  • :is()　は10点
  • :where()　は0点
  • どちらも詳細度は関係あるただ擬似クラスそのものに点数があるかどうかだけ異なる
