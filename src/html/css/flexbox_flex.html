Sample flexbox example

  • flex-grow - flex: 2; 割り当てられる空間の割合
  • flex-basis - flex: 30%;
  • flex-grow | flex-basis - flex: 1 200px; flex itemsの最小大きさ
  • flex-grow | flex-shrink - flex: 2 2; shrinkはflex-containerからはみ出るようになった時縮小させる
  • flex: 2 2 10%;
  • 単位がついていたらflex-basis,
    shrinkは　flex: 2 2;のようにflex-growの次に書くかflex-shrinkで書くしかない
  • flex だけの話ではないけど正式名称（flex-grow, flex-sh...)は以前の定義を再定義する場面以外は使わない方がいい（複雑になる）
